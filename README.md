# 🐚 Abalone Data Analysis | Exploratory Data Analysis Project

## 📌 Project Overview

This project focuses on performing exploratory data analysis (EDA) on the Abalone dataset to understand how different physical characteristics influence the age of abalones.

The primary objective is to analyze relationships between features such as size, weight, and rings (age indicator), and extract meaningful insights from the data.

---

## 📂 Dataset Information

The Abalone Dataset contains physical measurements of abalones. Key attributes include:

* Length
* Diameter
* Height
* Whole Weight
* Shucked Weight
* Viscera Weight
* Shell Weight
* Rings (used to estimate age)

* Heatmap for correlation analysis

---

## 📊 Key Insights

* Physical size (length, diameter) shows a strong relationship with age
* Weight-related features are positively correlated with rings
* Some features exhibit multicollinearity
* Outliers are present in height and weight measurements
* Age prediction using rings can be influenced by multiple combined factors
